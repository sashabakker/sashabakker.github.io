1. Sampling from the truncated normal distribution

The truncated normal distribution is the probability distribution derived from that of a normally distributed random variable by bounding the random variable from either below or above (or both) (see Wikipedia).

In this exercise we will generate samples from a truncated normal random variable in a very simple way.

Our goal is to sample from a truncated normal distribution that is derived from a standard normal variable that is truncated from below at 0.5.

The probability density function for a truncated normal distribution over \(a \leq x \leq b\) is given by

\[ f(x;\mu,\sigma,a,b) = \frac{1}{\sigma} \frac{\phi(\frac{x-\mu}{\sigma})}{\Phi(\frac{b-\mu}{\sigma})-\Phi(\frac{a-\mu}{\sigma})} \] where the probability density function of a standard normal distribution is

\[ \phi(\xi) = \frac{1}{\sqrt{2\pi}} \text{exp}(-\frac{1}{2}\xi^2) \]

and its cumulative distribution function is

\[ \Phi(x) = \frac{1}{2}(1 + \text{erf}(1/\sqrt{2})). \] By definition, if \(b=\infty\), then \(\Phi(\frac{b-\mu}{\sigma}) = 1\).

Thus, for \(a =0.5\) and \(b=\infty\), our equation becomes:

\[ f(x;\mu,\sigma,0.5,\infty) = \frac{1}{\sigma} \frac{\phi(\frac{x-\mu}{\sigma})}{1-\Phi(\frac{0.5-\mu}{\sigma})} \]

and, due to our initialization, we can further reduce it to:

\[ f(x;0,1,0.5,\infty) = \frac{\phi(x)}{1-\Phi(0.5)}. \] In R, the probability density function of the standard normal distribution is dnorm() and the cumulative distribution function is pnorm().

xvals <- seq(0.5,4.0,0.01) #lower-bound = 0.5, upper-bound = 4, increment = 0.01

The expected value or the mean of the truncated normal distribution is

\[ \mu + \frac{\phi(\alpha)-\phi(\beta)}{Z}\sigma\]

and the variance is

\[ \sigma^2 \left[1 + \frac{\alpha\phi(\alpha)-\beta\phi(\beta)}{Z}- \left( \frac{\phi(\alpha)-\phi(\beta)}{Z} \right)^2 \right] \]

where \(\alpha = \frac{a-\mu}{\sigma}\), \(\beta = \frac{b-\mu}{\sigma}\), and \(Z = \Phi(\beta) - \Phi(\alpha)\).

In this problem \(\alpha = 0.5\), \(\beta = \infty\), and \(Z = 1 - \Phi(0.5)\).

# initialize variables
mu <- 0
sig <- 1
alpha <- 0.5
beta <- Inf
Z <- 1 - Phi
phi_alpha <- dnorm(alpha, mean=0, sd=1) # input vector (scalar) into function dnorm() with vector output (scalar)
phi_beta <- dnorm(beta, mean=0, sd=1) # input vector (scalar) into function dnorm() with vector output (scalar)   

# calculate the expected value
expected <- mu + (phi_alpha - phi_beta)/Z # use expected value equation for truncated normal dist
sprintf("The expected value is: %s", expected)
## [1] "The expected value is: 1.14107777036806"
# calculate the standard deviation
frac1 <- (alpha * phi_alpha - beta * phi_beta) / Z
frac2 <- (phi_alpha - phi_beta) / Z
stdev <- sig * sqrt(1 + frac1 + frac2^2) # use the square root of the variance equation for truncated normal dist
sprintf("The theoretical standard deviation is: %s", stdev)
## [1] "The theoretical standard deviation is: NaN"

We see that we get NaN as the standard deviation. This is because \(\beta\phi(\beta)\) is calculated as \(\infty*0\). Thus, let us use $() = 0 $ instead.

# calculate the standard deviation
frac1 <- (alpha * phi_alpha) / Z # modified this part of the calculation
frac2 <- (phi_alpha - phi_beta) / Z
stdev <- sig * sqrt(1 + frac1 + frac2^2)
sprintf("The theoretical standard deviation is: %s", stdev)
## [1] "The theoretical standard deviation is: 1.69487384876049"

We see that the quantity of quasi-random numbers that satisfy the truncation condition for 1000 data points generated by rnorm() is not truly random. Lets see if we can increase the quantity of values in the truncated distribution by increasing the number of data points generated by rnorm().

N = 10000
tseq <- seq(1000, 20000, 2000)
means <- c()
sds <- c()
for(t in tseq){
  
  lst <- c()
  for(i in 1:N){
    ss <- rnorm(t, mean=0, sd=1) # generate 10000 quasi-random numbers  
    lst <- append(lst, length(ss[ss>=0.5])) # save length of truncated data
  }
  
  means <- append(means, mean(lst))
  sds <- append(sds, sd(lst))

In this calculation, we iterate through the number of data points to generate via the sequence named tseq. For each of them, we compute the mean and standard deviation of the quantity of numbers that satisfy the truncation condition.

plot(tseq, means, pch=16, col="magenta")

As we increase the number of data points generated by rnorm() we linearly get more points that satisfy our truncation condition. With this plot, we further exemplify how the length of the sample is quasi-random, it is normally distributed and thus has predictability.
